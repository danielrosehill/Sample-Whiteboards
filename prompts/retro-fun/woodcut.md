# Woodcut

**Model:** Fal AI Nano Banana 2 (image-to-image)  
**Endpoint:** `fal-ai/nano-banana-2/edit`  
**Style:** Medieval woodcut / linocut print

## Prompt

> Take this whiteboard photograph and convert it into a diagram styled as a medieval woodcut or linocut print. Remove the physical whiteboard, markers, frame, and any background elements -- output on an aged parchment or cream-colored background. Correct any perspective distortion so the output appears as a perfectly straight-on, top-down view. Preserve all the original content, text, and diagrams. When reading handwritten text, infer the correct spelling of technical terms, product names, and proper nouns rather than transcribing handwriting literally (e.g. "Proxmox" not "Proxknox", "Kubernetes" not "Kubernites"). Render all elements using bold black carved lines with visible wood-grain texture in the strokes. Use cross-hatching for shading and fills. Text should appear in a blackletter or old-style serif typeface. Shapes should have a rough, hand-carved quality. Optionally add a single spot color (a deep red or ochre) for emphasis on key elements. The result should look like a page from an illuminated technical manuscript -- ancient, authoritative, and delightfully absurd as a format for modern tech diagrams.

## Style Notes

- Think: Gutenberg Bible illustration, medieval manuscript, vintage textbook plate.
- Bold black lines with cross-hatching, visible carving texture.
- The contrast between ancient style and modern tech content is inherently entertaining.
- Great for blog posts with a humorous or historical angle.
