# Chalkboard

**Model:** Fal AI Nano Banana 2 (image-to-image)  
**Endpoint:** `fal-ai/nano-banana-2/edit`  
**Style:** Classic chalkboard / lecture hall

## Prompt

> Take this whiteboard photograph and convert it into a diagram styled as if drawn on a classic green chalkboard. Remove the physical whiteboard, markers, frame, and any background elements -- output the diagram on a dark green chalkboard-textured background with subtle chalk dust. Correct any perspective distortion so the output appears as a perfectly straight-on, top-down view. Preserve all the original content, text, and diagrams. When reading handwritten text, infer the correct spelling of technical terms, product names, and proper nouns rather than transcribing handwriting literally (e.g. "Proxmox" not "Proxknox", "Kubernetes" not "Kubernites"). Use white and colored chalk (yellow, pink, light blue) for all lines, shapes, and text. The lettering should look like clean chalk handwriting. Add subtle smudge effects and chalk texture to lines. The result should feel like a beautifully organized lecture -- academic, warm, and intellectual.

## Style Notes

- Think: university lecture, physics classroom, TED talk background.
- Green chalkboard with realistic chalk texture.
- Colored chalk for emphasis and grouping.
- Warm, academic, nostalgic feel.
